Sliding Puzzle

Main Menu

Objective

Solve a puzzle by rearranging tiles in correct order to make a complete image.

Rules

New Game: At the beginning of the game click New Game button to shuffle the tiles.

Tile Movement: The game is played on a grid with square tiles. There is one empty space that allows tiles to slide into it.

Tile Sliding: Tiles can be moved horizontally or vertically into the empty space. Only one tile can occupy the empty space at a time, and tiles cannot jump over one another.

Winning: The game is considered solved or won when the tiles are in the correct order to make complete image.

Reset and Shuffle: Players can reset the puzzle to its initial state by clicking New Game to shuffle the tiles if they want to start over.

Change Puzzle: Player can change image of puzzle by clicking on Menu button, then Change Puzzle to see options of images.

Change Level: Player can change level by clicking on Menu button, then Change Level. Options available are 3x3 or 4x4 grid.
